# ResumeForge – Smart Resume Builder

ResumeForge v4 is a modern, frontend-only **Resume Builder web application** built using **HTML, Bootstrap, CSS, and Vanilla JavaScript**.  
It helps users create professional resumes with **live preview**, **multiple templates**, **ATS score insights**, **dark mode**, **JSON backup/import**, and **PDF export** — all in a polished and responsive interface.

---

## 🚀 Live Project Overview

ResumeForge is designed as a **portfolio-grade frontend project** for MCA / web developer resumes.  
It allows users to fill in their resume details, instantly preview the resume, switch templates, export/import data, and generate a clean printable PDF.

This project is especially useful for:
- MCA / BCA / BE / BTech students
- Fresher web developers
- Anyone who wants to create an ATS-friendly resume quickly

---

## ✨ Features

### Core Resume Builder Features
- **Live Resume Preview** while typing
- **Tabbed form editor** for clean UX
- **3 Resume Templates**
  - Modern Sidebar
  - Minimal Clean
  - Professional Corporate
- **Profile Photo Upload**
- **Dynamic Add / Remove Sections**
  - Education
  - Projects
  - Experience / Internship
  - Certifications
  - Achievements
  - Languages

### Smart Features
- **ATS Score Panel**
  - Gives a basic resume quality score
  - Suggests improvements for better ATS readiness
- **Section Visibility Toggle**
  - Show / hide skills, projects, experience, certifications, achievements, and languages
- **Resume Completion Progress Bar**
- **Character Counters**
  - Summary
  - Project descriptions
  - Experience descriptions

### Productivity Features
- **Fill Sample Data** for instant demo content
- **Save Resume to LocalStorage**
- **Auto-load saved data on refresh**
- **Clear Resume**
- **JSON Export / Import**
- **Dark Mode / Light Mode**
- **Print Resume**
- **Download Resume as PDF**
